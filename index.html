О проекте
Два независимых кейса: анализ VK-кампании зоомагазина «Четыре Лапы» и оптимизация кампаний в Яндекс Директе для интернет-магазина кофемашин «Кофетека».
Что было
В VK три сегмента тратили бюджет без анализа CPO — разница между ними достигала 20 раз. В Директе кампании с наибольшим расходом давали CPA выше 1 500 ₽ без оптимизации.
Цель
Найти неэффективные сегменты VK, сформулировать гипотезы перераспределения бюджета. Оптимизировать кампании Директа: убрать нецелевой трафик, поднять CTR объявлений.
Действия
  • Проанализировал CPO по трём сегментам VK — нашёл разрыв в 20 раз (94 ₽ vs 1 940 ₽)
  • Сформулировал 3 измеримые гипотезы оптимизации с прогнозом достижения KPI
  • Рассчитал 9 метрик по 7 месяцам данных Директа, выявил рост ДРР как сигнал выгорания
  • Нашёл 2 убыточные кампании, собрал минус-слова, улучшил объявления с CTR < 7%
Результат
Выявлена единственная рентабельная связка VK с CPO в 2,5 раза ниже KPI. Для Директа: набор гипотез и конкретных действий по перераспределению бюджета и улучшению CTR.

О проекте
7 месяцев работы менеджером по продажам в сети фитнес-клубов. Личная инициатива: выстроил систему удержания и сегментированный подход к продлению абонементов.
Что было
30% клиентов не продлевали абонемент. Ресепшен и отдел продаж работали раздельно — клиент уходил после тренировки без единого касания.
Цель
Снизить отток с 30% до 10–15% и увеличить продажи через работу с сегментами ЦА — без дополнительного бюджета.
Действия
  • Выстроил 4-шаговую воронку удержания: ежедневная выгрузка базы → скрипт для ресепшена → момент касания → личная встреча
  • Разбил клиентов на сегменты: Елена (30+, похудение) и Андрей (24, набор массы)
  • Для каждого сегмента — свой аргумент: сообщество для Елены, 1 встреча с тренером для Андрея
  • Персональный оффер вместо шаблонных фраз — диалог строился от цели и боли клиента
Результат
Продляемость абонементов выросла на 15–20%, план продаж перевыполнялся на 5–10%. Воронка работала без доп. бюджета — только процесс и понимание клиента.
